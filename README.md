# CitaPro---proyecto
CitaPro es una plataforma digital para la gestión inteligente de citas y turnos, permitiendo a usuarios y negocios organizar reservas, horarios y servicios.

## Stack tecnológico

- Flutter
- Dart
- Supabase
- Git/GitHub
- Android Studio

## Integrantes

- Juan Oliver Ramos
- Nayely Núñez

## Arquitectura

El proyecto utiliza MVVM con principios de Clean Architecture.

La arquitectura permite separar la interfaz de usuario, la lógica de negocio y el acceso a datos, facilitando el mantenimiento y escalabilidad del sistema.
